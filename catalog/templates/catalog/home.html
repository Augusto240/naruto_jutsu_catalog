Sobre o Projeto

O Naruto Jutsu Catalog é um projeto desenvolvido como parte do meu portfólio, demonstrando habilidades em desenvolvimento web com Django e Python.

Este catálogo permite explorar os diversos jutsus do universo Naruto, oferecendo detalhes sobre cada técnica, incluindo seu elemento, tipo, rank e descrição completa.

O projeto implementa um CRUD completo, permitindo adicionar, visualizar, editar e excluir jutsus, além de oferecer uma API REST para integração com outros sistemas.

Sinta-se à vontade para explorar o catálogo e descobrir mais sobre as técnicas ninjas!
